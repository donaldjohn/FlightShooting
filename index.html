天空霸者

基于 Three.js 的 3D 飞行射击游戏

版本 1.0

技术栈

  • Three.js 0.155 - 3D 渲染引擎
  • Web Audio API - 程序化音效合成
  • WebGL - GPU 加速
  • localStorage - 用户数据持久化

特色

  • 所有 3D 模型和音效均由代码程序化生成
  • 零外部资源依赖（除 Three.js CDN）
  • 支持桌面端键鼠和移动端触屏
  • 4 个关卡 + BOSS 三阶段战斗
